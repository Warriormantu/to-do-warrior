About To-Do Warrior

To-Do Warrior is a simple and user-friendly task management web application designed to help users organize their daily activities efficiently.

This project is built using core web technologies like HTML, CSS, and JavaScript with a focus on clean DOM manipulation and beginner-friendly logic.

The goal of To-Do Warrior is to improve productivity by allowing users to add, manage, and remove tasks easily.
